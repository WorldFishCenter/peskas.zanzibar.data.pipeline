# Process catch data from surveys

Processes and structures catch data from surveys, including catch
numbers and associated vessel/gear information.

## Usage

``` r
process_catch_data(preprocessed_surveys, trips_info)
```

## Arguments

- preprocessed_surveys:

  A preprocessed survey dataframe

- trips_info:

  Trip information dataframe from extract_trips_info

## Value

A dataframe containing processed catch data
